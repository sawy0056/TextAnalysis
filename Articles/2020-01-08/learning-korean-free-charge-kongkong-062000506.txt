The popularity of K-pop has naturally led to interest in Korea. This makes it easy to find people traveling to Korea, watching Korean YouTube videos, and learning Korean. The number of people learning Korean has expanded dramatically. KONGKONG is a new application designed to help users learn Korean.KONGKONGMoreFor Indonesian users who want to learn Korean, KONGKONG can provide users with a quick an easy way to study. This is because users can study Korean through popular Korean videos. In particular, various videos of Korean dramas, music videos, and entertainment are updated daily, so it is possible to pick a favorite to help learn more Korean. Another great advantage is that users can read about Korean trends or learn buzzwords that fit different situations.In addition, there are no advertisements in the app, so anyone interested in Korean can use it without distractions from ads. It is a free application, but it has features such as repeat listening and grammar quizzes, so users can get a clear learning effect.
